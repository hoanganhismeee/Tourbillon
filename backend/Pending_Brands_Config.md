# Pending Brands Configuration Template

Use this file to paste the HTML elements and URLs for the remaining brands. Once filled, you can easily transfer the selectors to `backend/Configuration/brand-configs.json`.

**Instructions:**
1. Visit the brand's official website.
2. Navigate to a collection page (Listing Page).
3. Inspect a **Product Card** and copy the outer HTML element (Right click -> Inspect -> Copy OuterHTML).
4. Click into a watch **Detail Page**.
5. Inspect the elements for Reference Number, Price, Specs, etc., and copy their HTML.
6. Paste them below under the respective brand.
